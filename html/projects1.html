Project Overview

The College Event Management System is a web-based platform built using PHP and MySQL, designed to streamline the management of events within a college environment. It allows administrators to create, update, and delete events, while students can view event details and register online.

This system eliminates the traditional paper-based registration process, making the entire event lifecycle more efficient and organized.

Key Features

  • Admin dashboard to manage events
  • Real-time event listing
  • Event date & time tracking
  • Secure data handling with MySQL
  • Fully responsive design
  • Login system for both admins and students

Workflow / How it Works

  1. Admin logs in and creates an event (title, date, description)
  2. Students browse available events and register
  3. Admin can view participant list and edit/delete events
  4. System stores all data securely in MySQL database
  5. Event Scheduled in the official calender

Technologies Used

  1. Frontend: HTML, CSS, JavaScript
  2. Backend: PHP
  3. Database: MySQL
  4. Frameworks: Bootstrap for responsiveness
  5. Version Control: Git & GitHub
