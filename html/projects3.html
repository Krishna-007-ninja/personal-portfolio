Project Overview

The Leaf Disease Detection App is a mobile application designed to help farmers and agriculturists identify plant leaf diseases in real-time using artificial intelligence.

Built using React Native for cross-platform mobile UI and FastAPI for the backend, the system uses a trained machine learning model to classify diseases based on uploaded leaf images.

This app enables farmers to take proactive measures by instantly recognizing diseases.

Key Features

  • Upload leaf image and get instant prediction
  • Pre-trained ML model using TensorFlow
  • Mobile App UI for user-friendly interaction

Workflow / How it Works

  1. User opens the mobile app on Android/iOS device
  2. From the app, the user clicks a photo of a leaf or selects one from the gallery
  3. The image is sent to the FastAPI backend via a REST API call
  4. Backend loads the trained CNN (Convolutional Neural Network) model
  5. Model processes the image and predicts the disease class
  6. The prediction result is sent back to the app and shown on screen
  7. The app displays:
    • Disease Name (e.g., Leaf Spot, Blight)
    • Confidence Score (e.g., Leaf Spot â€” 93.8% confident)

Technologies Used

  1. Frontend: React-Native
  2. Backend: FastAPI (Python)
  3. Model: TensorFlow
  4. Image Input: Mobile Camera / Gallery
