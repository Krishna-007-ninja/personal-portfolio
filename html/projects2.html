Project Overview

The Spotify UI Clone is a fully responsive frontend project built using pure HTML and CSS that replicates the modern, sleek look of the popular Spotify music streaming app.

This clone includes essential UI components like a sidebar menu, navigation bar, music controls, playlists, and responsive cards â€” all designed without using any JavaScript.

Key Features

  • Sidebar navigation with icons
  • Playlist section with cover art and song titles
  • Footer with music player layout (play/pause, next/prev buttons)
  • CSS transitions and hover effects
  • Font Awesome icons integration

Technologies Used

  1. HTML5 for semantic structure
  2. CSS3 for layout and styling
  3. Font Awesome for icons
